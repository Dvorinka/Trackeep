How to get your API key:
  1. Log into your Trackeep account
  2. Go to Settings → Security
  3. Click "Generate New API Key"
  4. Copy the generated key (starts with tk_)
  5. Paste the key in the field above
  6. API keys are more secure than JWT tokens - they can be revoked anytime and have limited permissions
